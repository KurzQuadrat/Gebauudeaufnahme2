# Smoke-Test-Checkliste

Diese Checkliste dient als manueller Regressionstest für alle künftigen Refactoring-Schritte.
Vor und nach jedem Schritt durchgehen – die App muss sich danach identisch verhalten wie vorher.

## Prüfpunkte

- [ ] App lädt ohne Konsolenfehler
- [ ] Neues Projekt anlegen
- [ ] Geschoss anlegen
- [ ] Raum anlegen
- [ ] Fenster hinzufügen
- [ ] Tür hinzufügen
- [ ] Heizkörper hinzufügen
- [ ] Heizlast berechnen lassen
- [ ] Fotoanalyse auslösen, falls API-Key vorhanden
- [ ] Exporttext erzeugen
- [ ] JSON exportieren
- [ ] JSON wieder importieren
- [ ] Browser neu laden und prüfen, ob localStorage-Daten erhalten bleiben
- [ ] Visueller Vergleich nach CSS-Änderungen (Layout/Aussehen unverändert)
